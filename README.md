Para ejecutar el programa se utiliza make run, el cual compila de ser necesario y ejecuta el programa.

Para limpiar todos los archivos creados en ejecucion se utiliza make clean, que removerá los txt resultantes y los archivos compilados.

Las distintas instancias se seleccionan a traves de input, se asume que los usuarios estarán en la carpeta de Instancias/usuarios y los nodos en Instancias/nodos, se asume que los valores de tamaño, instancia y usuario serán correctos y existirá sus archivos correspondientes.
Ej: con tamaño 10, instancia 0 y 5 usuarios, se asume que existira un archivo llamado 10_instancia_0.txt en nodos y un archivo llamado 10_instancia_0_5us.txt en usuarios.
